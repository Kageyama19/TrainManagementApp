# Use Case 2: Add Passenger Bogies to Train

This use case demonstrates dynamic consist management using `ArrayList`.

## Implemented Flow
1. User runs the program.
2. Passenger bogies are added.
3. Bogies are displayed.
4. One bogie is removed.
5. Existence is checked with `contains()`.
6. Program continues with final list output.

## Java File
`src/TrainAppUC2.java`

## Sample Output
```text
=== Train Consist Management App ===
Use Case 2: Add Passenger Bogies to Train
Passenger bogies after addition: [Sleeper, AC Chair, First Class]
Removed bogie: AC Chair
Is Sleeper present? true
Final passenger bogie list: [Sleeper, First Class]
```
