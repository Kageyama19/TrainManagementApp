# Use Case 5: Preserve Insertion Order of Bogies (LinkedHashSet)

This use case demonstrates ordered and unique train formation using `LinkedHashSet`.

## Flow
1. User adds bogies.
2. `LinkedHashSet` stores them in insertion order.
3. Duplicate bogies are ignored automatically.
4. Final formation is printed.

## Java File
`src/TrainAppUC5.java`

## Sample Output
```text
=== Train Consist Management App ===
Use Case 5: Preserve Insertion Order of Bogies
Final train formation: [Engine, Sleeper, Cargo, Guard]
Total unique bogies: 4
```
